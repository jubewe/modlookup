Hey! Welcome to my page What is this Website / Bot? The Bot is a newly developed Tool that allows you to get different Stats of the Moderators and VIPs on the platform Twitch This works by tracking the badges of a user via chat messages (Yes, that is the easiest option (for me) since Twitch does not support the /mods Command for Non-Broadcasters anymore) The Bot is in some of the live channels (except for the permanent channels only German and English channels are logged at the moment) You can request a channel to be permanently logged via the /suggestchannel endpoint You can add the bot by using ml!join in the bots channel Chatbot Prefix: ml!
